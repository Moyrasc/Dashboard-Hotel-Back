Hotel Miranda

ENDPOINTS

  • POST /login
  • Bookings: Required Login
      • GET /bookings => return all bookings
      GET /:bookingId => return one booking
      POST / => create new booking
      PUT /:bookingId => edit a booking
      DELETE /:bookingId => delete a booking
  • Rooms: Required Login
      • GET /rooms => return all rooms
      GET /:roomId => return one room
      POST / => create new room
      PUT /:roomId => edit a room
      DELETE /id/:roomId => delete a room
  • Contacts: Required Login
      • GET /contacts => return all contacts
      GET /:contactId => return one contact
      POST / => create new contact
      PUT /:contactId => edit a contact
      DELETE /:contactId => delete a contact
  • Users: Required Login
      • GET /users => return all users
      GET /:userId => return one user
      POST / => create new user
      PUT /:userId => edit a user
      DELETE /:userId => delete a user
