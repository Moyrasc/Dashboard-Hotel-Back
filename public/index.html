Hotel Miranda

ENDPOINTS

  • POST /login
  • Bookings: Required Login
      • GET /bookings : return all bookings
      GET /id/:bookingId : return one booking
      POST /newBooking : create new booking
      PUT /editBooking/:bookingId : edit a booking
      DELETE /id/:bookingId : delete a booking
  • Rooms: Required Login
      • GET /rooms : return all rooms
      GET /rooms/id/:roomId : return one room
      POST /newRoom : create new room
      PUT /editRoom/:roomId : edit a room
      DELETE /id/:roomId : delete a room
  • Contacts: Required Login
      • GET /contacts : return all contacts
      GET /contacts/id/:contactId : return one contact
      POST /newContact : create new contact
      PUT /editContact/:contactId : edit a contact
      DELETE /id/:contactId : delete a contact
  • Users: Required Login
      • GET /users : return all users
      GET /users/id/:userId : return one user
      POST /newUser : create new user
      PUT /editUser/:userId : edit a user
      DELETE /id/:userId : delete a user
